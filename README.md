# ds4paul_ai
Repozytorium poświęcone projektom i materiałom z zakresu sztucznej inteligencji (AI), głębokiego uczenia (deep learning) oraz przetwarzania języka naturalnego (NLP).

## Zakres tematyczny

- Sieci neuronowe (ANN, CNN, RNN)
- Modele transformers i LLM (np. BERT, GPT, LLaMA)
- NLP: tokenizacja, klasyfikacja tekstu, analiza sentymentu
- Generatywne modele: tekst → obraz, tekst → tekst
- Transfer learning, fine-tuning modeli
- Praktyczne zastosowania AI (np. chatboty, klasyfikatory, systemy rekomendacyjne)
